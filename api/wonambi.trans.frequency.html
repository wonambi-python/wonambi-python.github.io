Compute power or energy acoss a frequency band, and its peak frequency. Power is estimated using the mid-point rectangle rule. Input can be ChanTime or ChanFreq.

Parameters:

  • data (instance of ChanTime or ChanFreq) ‚Äì data to be analyzed, one trial only

  • freq (tuple of float) ‚Äì Frequencies for band of interest. Power will be integrated across this band, inclusively, and peak frequency determined within it. If a value is None, the band is unbounded in that direction.

  • input_type (str) ‚Äì ‚Äòtime‚Äô or ‚Äòspectrum‚Äô

  • scaling (str) ‚Äì ‚Äòpower‚Äô or ‚Äòenergy‚Äô, only used if data is ChanTime

  • n_fft (int) ‚Äì length of FFT. if shorter than input signal, signal is truncated; if longer, signal is zero-padded to length

  • array_out (bool) ‚Äì if True, will return two arrays instead of two dict.

Returns:

  • dict of float, or ndarray ‚Äì keys are channels, values are power or energy

  • dict of float, or ndarray ‚Äì keys are channels, values are respective peak frequency

instance of ChanFreq ‚Äì If output=‚Äôcomplex‚Äô, there is an additional dimension (‚Äòtaper‚Äô) which is useful for ‚Äòdpss‚Äô but it‚Äôs also present for all the other tapers.

Create a Morlet wavelet.

Parameters:

  • freq (float) ‚Äì central frequency of the wavelet

  • s_freq (int) ‚Äì sampling frequency

  • ratio (float) ‚Äì ratio for a wavelet family ( = freq / sigma_f)

  • sigma_f (float) ‚Äì standard deviation of the wavelet in frequency domain

  • dur_in_sd (float) ‚Äì duration of the wavelet, given as number of the standard deviation in the time domain, in one side.

  • dur_in_s (float) ‚Äì total duration of the wavelet, two-sided (i.e. from start to finish)

  • normalization (str) ‚Äì ‚Äòwonambi‚Äô (default) returns an amplitude of 1 in frequency-domain for a sine wave of amplitude 1 in the time-domain; ‚Äòjuniper‚Äô returns amplitude which is dependent on sampling frequency; ‚Äòarea‚Äô normalizes the area of the Gaussian envelope to be 1; ‚Äòpeak‚Äô normalizes the peak of the Gaussian envelope to be 1. Note that the frequency-domain values for ‚Äòarea‚Äô and ‚Äòpeak‚Äô will depend on the carrier frequency of the wavelet (they depend on sigma_f).

  • zero_mean (bool) ‚Äì make sure that the wavelet has zero mean (only relevant if ratio < 5)

Returns:

ndarray ‚Äì vector containing the complex Morlet wavelets

Notes

‚Äòratio‚Äô and ‚Äòsigma_f‚Äô are mutually exclusive. If you use ‚Äòsigma_f‚Äô, the standard deviation stays the same for all the frequency. It‚Äôs more common to specify a constant ratio for the wavelet family, so that the frequency resolution changes with the frequency of interest.

‚Äòdur_in_sd‚Äô and ‚Äòdur_in_s‚Äô are mutually exclusive. ‚Äòdur_in_s‚Äô specifies the total duration (from start to finish) of the window. ‚Äòdur_in_sd‚Äô calculates the total duration as the length in standard deviations in the time domain: dur_in_s = dur_in_sd * 2 * sigma_t, with sigma_t = 1 / (2 * pi * sigma_f)
